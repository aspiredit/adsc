One-time device setup

To publish events to the website, this device needs a GitHub access key. This is a one-time step — once saved it stays on this device and you won't see this screen again.

How to get the access key:
  1. Go to GitHub → Settings → Fine-grained tokens
  2. Click Generate new token
  3. Under Repository access select aspiredit/adsc
  4. Under Permissions → Contents choose Read and write
  5. Generate, copy the token, and paste it below
